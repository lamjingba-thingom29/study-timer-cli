# Study Timer ⏱️

A simple command-line study timer with **pause**, **resume**, and **stop** controls.  
Automatically logs completed study sessions to a daily log file.

---

## Features

- Set a custom study duration
- Pause timer (`p`)
- Resume timer (`r`)
- Stop timer early (`s`)
- Automatically logs study time in `study_log.txt`
- Shows total study time completed across all sessions
